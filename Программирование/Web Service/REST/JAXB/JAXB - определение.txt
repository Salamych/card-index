--->JAXB (Java Architecture for XML Binding - архитектура Java для привязки XML) - это специальный инструмент 
для маршалинга и демаршалинга объектов. Это программная структура, которая позволяет разработчикам отображать 
классы Java в представления XML. JAXB позволяет делать сохранение объекта в файл и обратно в объекты. 

JAXB предоставляет аннотации, которыми размечаются поля JAVA-классов.

Принцип получается примерно следующий: если тег имеет дочерние элементы или атрибуты, значит это объект.
Если тег в документе не имеет дочерних элементов, значит это поле объекта.

What is Behavior Verification (проверка поведения)?

- Одним из преимуществ использования объектов Mockito в том, что после его создания он 
  запоминает все операции выполненные над ним.

- Проверка поведения помогает нам провеить, был ли определенный эмитационный метод
  вызван тестируемой системой или нет.

- Проверка поведения выполняется с помощью метода verify()

- Используя метод verify() проверяется, был ли вызван проверяемый метод один и только
  один раз с предоставленными аргументами.

- Метод verify() использует метод equals() для сравнения аргументов и проверки того,
  что эмитационный метод был вызван с правильными аргументами.

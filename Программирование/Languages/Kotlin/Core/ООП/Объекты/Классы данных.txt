Классы данных спроектированны специально для хранения данных и предлагают
широкие возможности для работы с данными.

Создается с использованием ключевого слова data.

package com.bignerdranch.nyethack

data class Coordinate(val x: Int, val y: Int){
    val isInBounds = x >= 0 && y >= 0
}

Классы данных предоставляют свои реализации функций toString, equalse, hashCode и т.д.,
а также поддерживают автоматическую деструктуризацию.

Можно вернуть свойства, объявленные в главном конструкторе:

val (x, y) = Coordinate(1, 0)

--------------Ограничения класса данных----------------------

Классы данных:

- должны иметь хотя бы один параметр в главном конструкторе;

- требуют, чтобы параметры главного конструктора объявлялись как var или val;

- не могут иметь модификаторы abstract, open, sealed, inner.

Ассоциативные массивы состоят из пар "ключ-значение", а вместо доступа
по целочисленному индексу предоставляют доступ по ключу указанного типа.
Ключи универсальны и определяют значения в ассоциативном массиве: значения,
напротив, не обязательно должны быть уникальными. С этой точки зрения
ассоциативные массивы обладают одним из свойств множеств: они гарантируют
уникальность ключей, подобно элементам множеств.

Создание ассоциативных массивов - mapOf  mutableMapOf

val patronGold = mapOf("Eli" to 10.5, "Mordoc" to 8.0, "Sothie" to 5.5)

val patronGold: Map<String, Double> = mapOf("Eli" to 10.5, "Mordoc" to 8.0, "Sothie" to 5.5)

Функция to преобразует операнды слева и справа в пару (Pair) - тип, представляющий группу из двух элементов.

Создание при помощи Pair

val partronGold  = mapOf(Pair("Eli", 10.75), Pair("Mordoc", 8.00), Pair("Sothie",5.50))

При добавление новой пары ключ-значение, если таковой ключ уже имеется в мапе, то он будет
перезаписан новым значением.

-------Часто применяемые функции доступа к элементам ассоциативного массива-----
[] - возвращает значение для указанного ключа или null, если ключа не существует

getValue - возвращает значение для указанного ключа, возбуждает исключение, если ключа не существует

getOrElse - возвращает значение для указанного ключа или значение по умолчанию, используя анонимную функцию

getOrDefault - возвращает значение для указанного ключа или заданное значение по умолчанию

----------Мутаторы ассоциативного массива------------------------

= - добавляет или обновляет значение для указанного ключа в массиве

+= - добавляет или обновляет элемент или элементы в массиве в зависимости от операнда справа,
     который может быть элементом или ассоциативным массивом 

put - добавляет или обновляет значение в массиве для указанного ключа

putAll - добавляет все пары "ключ-значение", переданные в аргументе

getOrPut - добавляет запись с указанным ключом, если она не существует, и возвращает результат;
	   иначе возвращает существующее значение

remove - удаляет запись из массива и возвращает значение 

- - возвращает новый массив без указанных записей 

-= - удаляет запись или записи из массива

clear - удаляет все записи из массива.

.toFloat()
.toDouble()
.toDoubleOrNull()
.toIntOrNull()
.toLong()
.toBigDecimal()

Так как попытка преобразования в разные числовые форматы может вызвать исключения,
Котлин предлагает функции безопастного преобразования toDoubleOrNull и toIntOrNull.
Если число нельзя преобразовать без ошибки, возвращается значение null вместо исключения.
Используем оператор ?: с toIntOrNull, например, чтобы вернуть другое значение:

val gold: Int = "5.91".toIntOrNull() ?: 0

Макет определяет набор объектов пользовательского интерфейса и их расположение на экране.
Макет формируется из определений, написанных на языке XML. Каждое определение используется
для создания объекта, выводимого на экране (например, кнопки или текста). (activity_main.xml)

По соглашению имя файла макета выбирается в зависимости от activity, с которой он связан: его
имя начинается со слова activity_, а остаток имени activity пишется строчными буквами с использованием
нижнего подчеркивания для разделения слов (стиль "змеиный регистр"). Файл макета для activity под
названием SplashScreenActivity.kt будет называться activity_splash_screen.xml
